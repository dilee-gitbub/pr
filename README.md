# AI 개인 브랜딩 랜딩 페이지

AI 활용이 막막한 직장인에게 실무에 바로 적용 가능한 AI 협업 방법을 제공하는 개인 브랜딩 모바일 랜딩 페이지입니다.

## 🎯 프로젝트 개요

"AI 활용이 막막한 직장인에게 실무에 바로 적용 가능한 AI 협업 방법을 통해, 업무 생산성을 극대화하고 창의적인 일에 집중하는 즐거움을 제공합니다."

## 📚 제공 콘텐츠

### 1. AI 워크 스마트
실전 AI 활용법 & 성공 사례를 통해 실무에 바로 적용할 수 있는 노하우를 배웁니다.

### 2. AI 동료 만들기
No-code/Low-code 업무 자동화로 반복적인 작업을 효율화하고 시간을 절약합니다.

### 3. AI로 데이터 인사이트 발견
데이터 분석 & 시각화를 통해 숨겨진 인사이트를 발견하고 의사결정을 개선합니다.

## 🛠 기술 스택

- **HTML5** - 시맨틱 구조
- **CSS3** - 반응형 디자인, 애니메이션
- **JavaScript** - 인터랙션, 스크롤 애니메이션

## 🎨 디자인 특징

- 밝은 파스텔 블루/민트 톤
- 모바일 우선 반응형 디자인
- 부드러운 페이드인/페이드아웃 애니메이션
- 카드 호버 효과 및 인터랙션

## 📱 반응형 디자인

- 모바일 (≤480px)
- 태블릿 (481px ~ 768px)
- 데스크톱 (≥769px)
